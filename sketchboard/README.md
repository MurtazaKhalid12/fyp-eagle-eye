# 🗒️ EagleEye Sketchboard

This is your **personal daily workspace**. Use it freely for experiments, quick tests, and ideas.

## Rules
- ✅ Copy files here from `model-training/` or `firmware/` before modifying them
- ✅ Experiment and break things freely — this folder is NOT production
- ❌ Don't put final, working code here — commit that to the proper folders
- ❌ Don't worry about cleaning up — it's a sketchboard!

## Structure

```
sketchboard/
├── model-training/     → Python training experiments
│   ├── train_tiny_model.py   (copy of the working model)
│   └── download_data.py      (copy of dataset downloader)
├── firmware/           → Arduino sketch experiments
└── notes/              → Quick text notes / ideas
```

## Workflow
1. Copy the file you want to experiment with into sketchboard/
2. Modify freely
3. If it works → copy the good version back to the real folder
4. If it doesn't → no harm done!
